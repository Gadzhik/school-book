# Универсальная читалка-конвертер для школьников

Офлайн-первое приватное приложение для чтения электронных книг. Единое
веб-ядро + тонкие платформенные оболочки (PWA → десктоп Tauri 2 → мобилки
Tauri 2 mobile) + опциональный библиотечный сервер для школы. Данные хранятся
на устройстве: ни телеметрии, ни облака. ИИ-функции работают через локальную
модель (Ollama / LM Studio) — без отправки текста наружу.

## Структура монорепо

```
packages/
  core/          # бизнес-логика без UI: модель документа, библиотека, хранилище,
                 #   классификация, «Мои слова», закладки/выделения, LLM-помощник
  reader-engine/ # обёртка над foliate-js (+ его pdf.js): рендер, пагинация, TOC,
                 #   прогресс, TTS, аннотации/выделения
  converters/    # конвертация в EPUB: HTML/MD/TXT/DOCX/RTF/ODT, PDF→текст, OCR (Tesseract)
  book-scanner/  # «книга из фото»: захват, авто-обрезка (OpenCV/jscanify), сборка CBZ/PDF/EPUB
  adapters/      # платформенные адаптеры: камера, нативный TTS (Tauri)
  network/       # клиент-серверный режим: OPDS, синхронизация, аккаунты, пэйринг
  ui/            # общие Svelte-компоненты, темы, сторы состояния
apps/
  web/           # PWA: Vite + Service Worker + manifest
  desktop/       # Tauri 2
  mobile/        # Tauri 2 mobile
  server/        # библиотечный сервер: Rust (axum) + SQLite + OPDS
assets/          # офлайн-словари (ru), шрифты (OpenDyslexic)
```

## Технологии

- **UI:** Svelte 5 + Vite
- **Движок чтения:** [foliate-js](https://github.com/johnfactotum/foliate-js) (EPUB/FB2/MOBI/AZW3/CBZ) + встроенный pdf.js (PDF)
- **Хранилище:** IndexedDB + OPFS (через `idb`)
- **Конвертация/OCR:** marked, mammoth, pdf.js / mupdf-wasm, Tesseract.js, KaTeX (формулы)
- **Оболочки:** Tauri 2 (десктоп и мобилки)
- **Сервер:** Rust + axum + rusqlite (SQLite), JWT + argon2, mDNS, OPDS
- **Монорепо:** pnpm workspaces + Turborepo

## Возможности

- Чтение **EPUB / FB2 / FB2.zip / PDF / MOBI / AZW3 / CBZ**; формат определяется по сигнатуре файла.
- Типографика: размер шрифта, интервал, поля, выравнивание, шрифт (засечки/без/дислексия OpenDyslexic); темы светлая/тёмная/сепия/контраст; режим e-ink; линейка чтения.
- Библиотека в IndexedDB + OPFS, восстановление позиции, прогресс, постоянное хранилище.
- Классификация по классам/предметам/категориям: фасетный фильтр, смарт-полки, авто-разметка (метаданные/имя файла/LLM).
- Озвучивание (Web Speech с подсветкой слова + нативный системный TTS), словарь по тапу, «Мои слова» с интервальным повторением.
- **Закладки и выделения с заметками**, оценка читаемости, геймификация (серия чтения), отчёт для учителя/родителя.
- ИИ-помощник (локальная модель): «объяснить просто», краткое содержание, квизы на понимание, пост-обработка OCR.
- **Создание книги из фотографий** страниц: авто-обрезка/перспектива, улучшение, OCR, сборка в CBZ / searchable PDF / перетекаемый EPUB.
- Конвертация документов (DOCX/RTF/ODT/HTML/MD/TXT) в EPUB; рендер формул KaTeX.
- **Клиент-серверный режим:** обнаружение сервера в локальной сети (mDNS), OPDS-каталог, скачивание книг офлайн, синхронизация прогресса/слов/закладок между устройствами.
- **Аккаунты и роли** (администратор / старший пользователь / учитель / ученик): регистрация и одобрение, назначение книг классам, задания с дедлайнами и отчётом, журнал действий и резервная копия.

## Запуск

```bash
pnpm install

# Веб-приложение (http://localhost:5173)
pnpm web:dev
pnpm web:build && pnpm web:preview   # прод-сборка (Service Worker активен только в проде)

# Десктоп (Tauri 2)
pnpm --filter @reader/desktop dev
pnpm --filter @reader/desktop build

# Библиотечный сервер (Rust) — см. apps/server/README.md
cd apps/server && cargo run
```

Мобильная сборка (Android) — см. `apps/mobile/README.md`. Сервер и его API —
`apps/server/README.md`.
